# CI-Triage-Env

OpenEnv RL environment for training LLMs to investigate ambiguous CI failures with verifiable rewards.

See [plan/README-PLAN.md](plan/README-PLAN.md) for the full build plan and per-branch task folders under [plan/](plan/).

## Quickstart

```bash
uv sync --all-extras
uv run pytest -q
```
